jquery-simpleTabs
=================

Really simple tabs, no css need

USAGE Example:

    <ul class="tabs" id="tabs">
      <li class="tab-test1 active"><a href="#">Tab 1</a></li>
      <li class="tab-tab2"><a href="#">tab 2<a></li>
      <li class="tab-lol"><a href="#">tab 3</a></li>
    </ul>
    <div id="tab-test1">
     content1
    </div>
    <div id="tab-tab2">
      content2
    </div>
    <div id="tab-lol">
      content3
    </div> 

init:
`$('#tabs').simpleTabs();`
